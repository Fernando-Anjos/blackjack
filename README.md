🃏 Blackjack Terminal Game (Ruby)

A simple terminal-based Blackjack game built in Ruby.
The project simulates a playable casino-style Blackjack experience with betting mechanics, dealer AI, and basic game flow.

🎮 Features

    ● Full Blackjack game loop (player vs dealer)
    ● Card drawing system with random deck logic
    ● Betting system with win/loss tracking
    ● Double down mechanic
    ● Dealer automatic play logic (hits until rules are met)
    ● Simple terminal interaction

🧠 Game Rules (Simplified)

    ● Try to get as close to 21 as possible without exceeding it
    ● Dealer must draw until reaching a minimum threshold
    ● Player can:
        ● Hit (draw a card)
        ● Stand (end turn)
        ● Double (double bet and draw one final card)

▶️ How to Run

Make sure you have Ruby installed.

bash:
ruby BlackJack.rb
